# Assignment 3: Classification Models and Evaluation Metrics

## Course
MBAI 5310G - AI Programming

## Project Overview
This project trains and compares two classification models to predict whether a business account will upgrade its account or not.

## Dataset
The dataset used is `business_account_upgrade_prediction_dataset.xls`.

## Target Variable
The target variable is `Upgraded_Account`.

## Models Used
The two classification models used are:

1. Logistic Regression
2. Support Vector Machine (SVM)

## Evaluation Metrics
Both models are evaluated using:

- Confusion matrix
- Accuracy
- Precision
- Recall
- F1-score

## Business Interpretation
The results are compared to identify which model performs better. The notebook also explains false positives, false negatives, model limitations, and why human judgment should still be used.
